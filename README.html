Layer Metadata

The standard layers plugin shows symbology. This plugin adds an alternative layer dock which is optimised for basic metadata access

You can group layers by a number of criteria such as type, provider or CRS

You can then drill down into basic layer metadata fields such as proj4, extent, field info and copy these to the clipboard with Ctrl-C

Open the dock using View > Panels > Layer Metadata

A dialog based version is also available using the toolbar icon... it's not as fully-featured, but gives much more space on smaller screens.
